All of our analysis is based on datasets downloaded from Kaggle and texts scraped from Wikipedia pages of Countries, Sports, and Athelets from the Olympic 2016. Below is presented an exciting example of what a Wikipedia page of a random athlete participating at the Olympic 2016 looks like.


We want to create an Olympic Network of the Countries, Sports, and Athletes to see how they connect. So therefore all the Wikipedia pages of Countries, Sports, and Athletes (if the athlete has a Wikipedia page) were downloaded. These Wikipedia pages will act as nodes in the network and the links between the Wikipedia pages will act like links in the network. A visualization of the network can be seen below.

Get an understanding of the Olympic Network

We want to dive deeper into the network and calculate some basic statistics..

It is important to get to know the network by its basic statistics and structure. This will give you a nice feeling, that you will know which type of network you are working with and how nodes are connected in the network. The Olympic Network is a directed network, which means that node A can link to node B, but node B does not necessarily link to node A.

The Olympic Network has 5.463 nodes and 16.306 links

We want to investigate the properties of the nodes. Let's start by finding links in and out of each node, this is also called the degree. The average degree of the total Olympic network is 5,97. The average is sensitive to large in and/or out degrees. Therefore it is interesting to study the median for the in- and out-degrees. The median is stable to nodes with extreme degrees. The median for in-degree is 2, which tells us that most nodes have very few in-degrees. For out-degree, the median is 0, which means that most of the nodes are not linking to other nodes.

Okay, so the network has many nodes with a low degree and very few with a high degree. Which corresponds to a power law and that the network has a hub-structure. A set of nodes where almost every node or at least many of the nodes link to is called a hub. This network structure is the main property in many real-world networks and it is also called a Scale-free Network. By having this network structure knowledge, further analysis will be more interesting and hopefully more successful.
